# Bonj menu image prompt set

Mode: built-in ImageGen
Use case: product-mockup

## Shared direction

Photorealistic premium editorial cafe product photography on a seamless warm
parchment-to-taupe studio backdrop. Centered three-quarter close view, complete
single serving visible, generous breathing room and landscape menu-card crop.
Soft directional window light from upper left, controlled natural shadow and
rich appetizing contrast. Cohesive modern Bonj campaign language. No people,
hands, text, logos, packaging, watermarks, busy props, illustration, CGI or
plastic-looking textures.

Cheesecakes use one minimal matte dark espresso-plum ceramic plate. Hot coffee
uses the same espresso-plum ceramic family; cold drinks use clear glass with
realistic ice and condensation.
